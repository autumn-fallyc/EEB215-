下載專案：將此GitHub的Repository完整下載並將資料夾解壓縮。
此專題需使用ultralytics (YOLOv8核心套件)與opencv-python (影像處理套件) 請在終端機執行以下程式
pip install ultralytics opencv-python
將此專題中的app.py和best.pt放置同一資料夾
最後利用終端機執行app.py即可

若想偵測影片 source="video.mp4"
